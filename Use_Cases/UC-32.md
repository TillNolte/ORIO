# UC-32 Gerät einer aktiven Kampagne zuordnen und Wechsel historisieren

Akteur: Operator

Ziel: Gerät genau einer aktiven Kampagne zuordnen; Wechsel möglich; Historisierung.

Regeln:

- Pro Gerät ist genau eine aktive Kampagnenzuordnung erlaubt
- Beim Wechsel in eine neue Kampagne wird die bisherige aktive Zuordnung automatisch beendet
- Jede Änderung der Zuordnung wird historisiert

Historie enthält mindestens:

- Gerät
- Kampagne
- Startzeitpunkt
- Endzeitpunkt
- Operator

Ergebnis:

- Aktive Kampagne je Gerät ist eindeutig
- Frühere Zuordnungen bleiben nachvollziehbar

Deckt ab: US-67
